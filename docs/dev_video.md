
## Gemini Video API Documentation（VEO 3.1）
根据以下文档，制作video-generation skills

``` 
## quick start
https://ai.google.dev/gemini-api/docs/quickstart
## user guide
https://ai.google.dev/gemini-api/docs/video?example=dialogue 
```
**requirement**
- image-generation skills 的执行流程很鲁棒，请参考 image-generation skills 的代码实现 video-generation skills，只需要留下python即可
- user guide文档很长，请尽量拆分成多个小节
- Veo prompt guide的内容需要整合进来,对于提高视频生成质量非常有价值
- 默认参数选择veo-3.1-fast, 720p, 4s
