# Vision Skills

本仓库汇总用于视觉相关Agent Skills
- 图像生成/编辑：基于 Gemini (Nano Banana)的文生图、图像编辑、多图合成
- 视频生成（占位，待完善）

**配置环境变量**
由于 Gemini 相关技能需要 API Key，请在使用前配置环境变量：
```bash
export GEMINI_API_KEY="YOUR_KEY"
```
